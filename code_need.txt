가상환경 활설화 코드
deactivate
비활성화 시키고 원래 기본 시스템 Python 환경으로 돌아간다.

깃허브에 올리는 명령어 (로컬에 수정된 파일이 있으면)
git add .
git commit -m "커밋할 내용"
git push 저장소이름 브랜치이름

깃허브 자체에서 수정된 파일이 있다면 (원격에 이미 올린 README나 다른 파일도 살리고 싶다면)
git pull 저장소이름 브랜치이름 --rebase
git push 저장소이름 브랜치이름 # 그 다음 다시 push 진행
